## LoginPage

```tsx
import { LoginPage } from "@sawabona/auth-kit";

export default function Login() {
  return <LoginPage />;
}
```

### Como funciona

- Faz `signIn("credentials")` via NextAuth
- Recebe config do `AuthProvider` para cores, redirecionamento e providers
- Mostra mensagens de erro automaticamente
- Suporta login via Google, GitHub, etc.
