# 🐘 @sawabona/auth-kit

**Pacote oficial de autenticação da Sawabona Tech para aplicações Next.js.**

Oferece tudo o que você precisa para proteger suas aplicações com segurança, escalabilidade e personalização total.

## Recursos principais

- 🔐 Autenticação com NextAuth.js
- 🧩 Componentes de UI prontos (LoginPage, RegisterPage)
- ✅ Tipagem forte com Zod
- 🎨 Customização via configuração
- 🚀 Integração com Supabase, Prisma ou API própria
- 🧪 Suporte a testes automatizados

## Para quem é

Ideal para devs que constroem:

- Aplicações SaaS multi-tenant
- Portais de cliente/admin
- Ferramentas internas protegidas
