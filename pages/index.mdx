# @sawabona/auth-kit

Autenticação plugável, segura e customizável para aplicações React e Next.js.  
Com suporte a múltiplos providers (NextAuth), layout adaptável, tema via TweakCN e lógica desacoplada.

### ✨ Principais recursos

- UI pronta com múltiplos layouts
- Autenticação por `credentials` ou qualquer provider OAuth
- Validações com Zod
- Tema 100% customizável via `themeTokens`
- Estrutura desacoplada, baseada em hooks
- Fácil de integrar a Supabase, Directus ou sua própria API
