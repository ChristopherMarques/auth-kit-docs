# ⚙️ Configuração do AuthProvider

```tsx
<AuthProvider
  config={{
    baseUrl: "https://api.exemplo.com",
    providers: ["credentials", "google", "auth0"],
    redirects: {
      afterLogin: "/painel",
      afterLogout: "/",
    },
    validation: {
      register: z.object({
        email: z.string().email(),
        password: z.string().min(8),
      }),
    },
    themeTokens: {
      "--primary": "#716C4A",
      "--font-sans": "Yeseva One, serif"
    },
    ui: {
      variant: "split-left"
    },
    branding: {
      logoUrl: "/logo.svg",
      companyName: "Sawabona Tech"
    }
  }}
>
```
