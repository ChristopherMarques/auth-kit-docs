# 🎨 Temas com TweakCN

Personalize completamente sua aplicação colando variáveis CSS do site [TweakCN](https://tweakcn.com)

```ts
themeTokens: {
  "--primary": "#716C4A",
  "--font-sans": "Yeseva One, serif",
  "--background": "#EFE9DB"
}
```

O hook `useThemeTokens` aplica os tokens diretamente em tempo de execução no `<html>`.

Você pode usar dark/light ou tokens condicionais.
